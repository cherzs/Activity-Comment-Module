Activity Comments

This module allows users to add comments to activities. Comments can be added, resolved, and managed within activities.

Learn More

Features

This module enhances the activity management system by allowing users to add detailed comments to activities. Users can add comments, mark them as resolved, and effectively manage all activity-related communications in one place. This feature improves collaboration and communication within the team while maintaining a clear record of activity-related discussions.

Benefits

  • Enhanced Communication: Enables detailed discussions and feedback directly within activities.
  • Better Organization: Keeps all activity-related communications in one place.
  • Improved Collaboration: Facilitates team collaboration through structured commenting system.

Installation

  1. Download the module from this repository.
  2. Install the module in your Odoo instance.

Compatibility

Odoo version: 16.0
